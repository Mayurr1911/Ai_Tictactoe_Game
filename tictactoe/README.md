# Tic-Tac-Toe AI (Beginner)

## Features
- **Unbeatable AI** using Minimax + Alpha-Beta pruning
- Smooth pygame UI (60 FPS)
- Human (X) vs AI (O)
- Press R to restart

## Run
```bash
cd games/tictactoe
python main.py
```

## What You Learn
- **Minimax algorithm**: Recursive game tree search
- **Alpha-Beta pruning**: Optimize by cutting branches
- **Game theory**: Perfect play leads to draw/tie

AI **never loses** - draws at best play!
